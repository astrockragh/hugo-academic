See this video [here](https://youtu.be/HBluLfX2F_k?si=AcBoaBPUKagxF7nY&t=2350), especially the end on why industries dominated by break-throughs should prioritize persistence over consistence. This seems relevant for science, and we should try to model it. Rewatch the video first, and see how science is a pursuit governed by power laws, pulling up citation metrics.

However, we should make the point that consistency is still somewhat important, especially since the counter is that you need to be able to stick around for long enough to actually still be a part of the scientific community, which drives the reasonable behaviour of any one person the other way. This can be seen as an example of the, "stick around for long enough to snowball"-effect mentioned in the video.
